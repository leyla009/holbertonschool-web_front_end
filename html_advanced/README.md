# HTML Advanced

## Description
This directory contains advanced HTML projects focused on semantic structure, document metadata, and best practices for modern web development. The goal is to build the foundation for a fully functional web application (HBnB) using clean and compliant HTML5.

## Projects

### 0. Create your first webpage
**File:** `0-index.html`
The first step in the project involves setting up a basic HTML5 skeleton.
- Includes the `<!DOCTYPE html>` declaration.
- Uses the `<html>` tag with `lang` (English) and `dir` (left-to-right) attributes.
- Serves as the starting point for a blank web document.

## Learning Objectives
By the end of these exercises, the following concepts are explored:
- How to structure an HTML document correctly.
- The importance of the `head` tag for metadata and resources.
- Using semantic tags to improve accessibility and SEO.
- Creating lists, links, and embedding media (images, iframes).
- Differentiating between visual formatting and semantic importance.
